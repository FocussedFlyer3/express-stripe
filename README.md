# Express Stripe

##### Tech Stack
- node.js
- express.js
- cors

## Install the dependencies
```bash
npm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
npm run dev
```
> Once run, should be available to access at http://localhost:4000

### Compiles and minifies for production

```
npm run build
```
